# 第一章 雨夜入城

次日，沈临川来到青石城。

青石城外的官道还带着昨夜的泥水，城门却已经提早打开。沈临川抬头看了一眼城楼上的黑旗，知道黑水盟的人比传闻里来得更早。

他这次不是路过。

三日前，玄灯司委托他调查黑水盟潜入城中的线人，并找回失踪的青铜令。那枚青铜令原本该锁在青石城府库，如今却在一夜之间失了踪影。

“如果令牌已经出城，就来不及了。”沈临川低声说道。

他没有直接进城，而是先去了城南的旧茶楼。旧茶楼里坐着一个名叫柳十二的说书人，明面上靠卖故事糊口，暗地里却替玄灯司递消息。

柳十二看着他，神色疲惫。

“你来晚了半日。”柳十二把一张被雨水打湿的地图推到桌边，“昨夜有人进了库房，带走青铜令的人，往黑水营方向去了。”

沈临川没有立刻接话。

他注意到地图角上有一道极细的刀痕，像是有人故意留下的记号。柳十二也看见了，却装作没看见。

“你在怕谁？”沈临川问道。

柳十二沉默了很久，才压低声音：“不是怕人，是怕名字。”

茶楼外忽然传来马蹄声，紧接着是短促的喝令。柳十二脸色一变，猛地站起身：“走后门，快！”

沈临川握住桌上的地图，转身离开。

# 第二章 黑水营的火

当夜，沈临川回到黑水营。

黑水营扎在青石城北面的河滩上，营门外燃着高火，风里全是湿木和铁锈的味道。他本想悄悄潜入，却在靠近军帐时看见了一个不该出现在这里的人。

柳十二。

柳十二正站在主帐外，低头对一个披黑甲的女人说话。那女人名叫顾照霜，是黑水盟在北地最凶狠的执事之一。传闻她从不留活口。

“原来你不是怕名字。”沈临川停在暗处，心里忽然明白过来，“你就是那个名字的一部分。”

他还没来得及继续靠近，脚下湿草忽然陷了半寸。

下一瞬，顾照霜抬起头，目光直直望向他藏身的方向。

“既然来了，”她的声音不高，却像刀刃擦过冰面，“就把地图和命，一起留下。”

沈临川没有后退。

他必须救出还被关在营中的线人，也必须弄清青铜令为什么会落到黑水盟手里。但他更在意另一件事:

地图上那道刀痕，和他师父当年留下的记号，一模一样。
